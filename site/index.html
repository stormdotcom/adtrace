How it works

Get started in under a minute. No accounts, no config, no servers.

1

Install the extension

Add AdTrace from the Chrome Web Store or load it as an unpacked extension for development.

2

Browse normally

AdTrace silently blocks ads and trackers using built-in filter lists. The popup shows live stats.

3

Open DevTools panel

Open the AdTrace DevPanel for full request-level tracing, rule attribution, and the tracker graph.
