Firefox Compatibility
AdTrace is built with Firefox MV3 compatibility in mind. The declarativeNetRequest API is supported in Firefox 113+. To install on Firefox: package as .xpi and submit to AMO, or load temporarily via about:debugging. The onRuleMatchedDebug listener requires Firefox Developer Edition or Nightly builds for full debug logging.
